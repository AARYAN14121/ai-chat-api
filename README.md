🤖 Gemini Chatbot API (FastAPI)

This is a simple AI chatbot backend built using FastAPI and Google Gemini API.  
It allows users to send messages and receive AI-generated responses.

- ⚡ FastAPI backend  
- 🤖 Gemini AI integration (`google-genai`)  
- 📩 REST API endpoint (`/chat`)  
- 🔐 Secure API key using `.env`  
- 🧠 Handles user prompts and returns AI responses  

Tech Stack
- Python 3.12  
- FastAPI  
- Uvicorn  
- Google GenAI SDK  
- Pydantic  
- Python-dotenv
